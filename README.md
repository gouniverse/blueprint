This is a quick start blueprint for an MVC web applications

- Redy to develop in the cloud (Gitpod / Github CodeSpaces)
- Router setup
- Controllers setup
- Database connection setup (SQLite example)
- CMS setup

## Installation

```
git clone https://github.com/gouniverse/blueprint
```

## Local Development

```
task dev:init
```

```
task dev
```

## Testing

```
task test
```

## Coverage Report

```
task cover
```
